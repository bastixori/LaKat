# LaKat 🐱💕

Una hermosa página web dedicada a los gatos Nevelung, desarrollada con amor para mi hermana.

## 📋 Descripción

LaKat es un sitio web elegante y moderno que celebra la belleza y carácter de los gatos Nevelung. La página presenta información detallada sobre esta raza felina de origen ruso, sus características, cuidados y mucho más.

## 🎨 Características

- **Diseño Responsivo**: Adaptable a todos los dispositivos (desktop, tablet, móvil)
- **Tema Rosado Elegante**: Paleta de colores cuidadosamente seleccionada
- **Logo Personalizado**: Ilustración SVG de un gato Nevelung en rosado
- **Contenido Completo**: Secciones informativas sobre la raza
- **Formulario de Contacto**: Interactividad con validación
- **Animaciones Suaves**: Transiciones y efectos visuales agradables
- **Navegación Fluida**: Scroll suave entre secciones
